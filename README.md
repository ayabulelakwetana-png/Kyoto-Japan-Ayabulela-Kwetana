# My Favorite City: Kyoto

This is a simple, responsive webpage created as part of the Code Your Future (CYF) Introduction to Digital course.

## Project Overview
The webpage showcases facts and images about Kyoto, Japan. It contains:
- At least 100 words of descriptive text.
- Three images (one of which is centered).
- A numbered list (ordered list).
- Responsive CSS styling.

## Deployment
This site is hosted on GitHub Pages.
